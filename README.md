# php_crud
Script sederhana untuk melakukan CRUD (Create, Read, Update, Delete) data mahasiswa menggunakan PHP dan MySQL.

Konfigurasi database MySQL yang diperlukan:
    <br>+ Server = localhost
    <br>+ User = root
    <br>+ Password = [kosong]
    <br>+ Nama Databse = crud

Konfigurasi tabel
  <br>nama tabel = mahasiswa
    <br>(nama kolom) (tipe data) (attribut tambahan)
    <br>+ id (int) AUTO_INCREMENT
    <br>+ nama (varchar)
    <br>+ nim (int)
    <br>+ alamat (varchar)

Anda bisa mengubah konfirgurasinya melalui file config.php
